Identyfikatory — ID

Pojęcie Znaczenie pojęcia (uzupełnia uczeń)
Co znaczy skrót ID? Identyfikator
Jaki jest cel stosowania ID? Do zróżnicowania podobnych elementów.
Jakie jest znaczenie znaku #? Oznacza początek definicji identyfikatora w CSS.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? id="nazwa_identyfikatora" - Atrybut 'id' w znaczniku HTML z unikalną nazwą.
Uwagi na temat stosowania ID (trzy konieczne wyliczenia)
  1. Identyfikator może wystąpić tylko raz w całym dokumencie.
  2. Elementy oznaczone ID muszą być unikatowe.
  3. Możan używać identyfikatorów wiele razy, ale jest to błędem. Nie wpływa to jednak na poprawność działania strony.
